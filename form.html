Description

This form allows you to generate and validate LLM (Large Language Model) instructions using a structured schema. Configure creativity, output, safety, and performance parameters for your AI models in a user-friendly way with GenAiInstructions.

Cognitive Level

Select the cognitive level required for your instruction, based on Bloom's taxonomy:

Remember

Recall of information

Example: "List Newton's 3 laws"

Understand

Reformulation and explanation

Example: "Explain the Product-Market Fit principle"

Apply

Direct implementation

Example: "Use this method to solve this customer case"

Analyze

Decomposition and structure

Example: "Compare these two pricing strategies"

Evaluate

Critical judgment

Example: "What is the weakness of this approach?"

Create

Original creation

Example: "Propose a new onboarding model"
